# MMNTUM Homebrew Tap

```bash
brew tap mmntum-llc/tap
brew install mmntum
```

`Formula/mmntum.rb` is regenerated by `make release VERSION=vX.Y.Z` in
[mmntum-cli](https://github.com/mmntum-llc/mmntum-cli) — binaries are hosted
on ShiftStack storage (https://shiftstack.ai/dl/mmntum/...). Don't edit the
formula by hand.
